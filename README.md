# Славянский лубок: Народный комикс

Мультимедийный лонгрид, созданный в рамках учебной практики. 

## Требования ТЗ, которые были реализованы:
- **Тема:** Славянский лубок (актуальность: возвращение к корням, анализ первого медиа).
- **Мультимедийность:** Текст, фото (в т.ч. галерея), видео-фрагмент (с YouTube), аудио (интершум ярмарки).
- **Архивы:** Использованы материалы (с указанием) из фондов Отдела эстампов Российской национальной библиотеки (Санкт-Петербург).
- **Интерактивность:** Слайдер «Было/Стало» (для демонстрации раскрашивания контура), интерактивный таймлайн истории жанра, фоновое аудио с кнопкой управления.
- **Адаптивность:** Корректное отображение на компьютерах и смартфонах (использован CSS Flexbox/Grid и media queries).
- **Нелинейность:** Наличие плавающего меню навигации по главам.

## Как опубликовать на GitHub Pages:
1. Создайте новый публичный репозиторий на вашем аккаунте GitHub.
2. Загрузите в него все файлы из этой папки (`index.html`, папки `css` и `js` со всем их содержимым).
3. Перейдите в настройки репозитория (**Settings**).
4. В левом меню выберите раздел **Pages**.
5. В разделе **Build and deployment**, под **Source** выберите `Deploy from a branch`.
6. Под **Branch** выберите `main` (или `master`) и папку `/ (root)`, затем нажмите **Save**.
7. Через несколько минут ваш сайт будет доступен по адресу вида `https://[ваш_логин].github.io/[имя_репозитория]/`.
